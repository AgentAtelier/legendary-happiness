# AUDIT-04 — Test Health

Severity: Critical = correctness/wrong-confidence bug; High = false-green, missing coverage on a critical path, real flakiness; Medium = coverage gap, brittle/non-discriminating assertion, marker mismatch; Low = nits/style.
Scope: foundry/tests/ (≈22 982 LOC, ≈50% of codebase). The test suite IS in scope this round. The.
Central question: do the tests verify the real behaviour, or do they pass while the build is broken?
